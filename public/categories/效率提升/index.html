Markdown 简介

Markdown 是一种轻量级标记语言，让你专注于内容而不是格式。

基础语法

标题

使用 # 创建标题，几个 # 就是几级标题。

强调

  • 粗体：使用 **文字**
  • 斜体：使用 *文字*
  • 删除线：使用 ~~文字~~

列表

无序列表：

  • 项目 1
  • 项目 2
    • 子项目 2.1
    • 子项目 2.2

有序列表：

  1. 第一步
  2. 第二步
  3. 第三步
