## Markdown 简介

Markdown 是一种轻量级标记语言，让你专注于内容而不是格式。

## 基础语法

### 标题

使用 `#` 创建标题，几个 `#` 就是几级标题。

### 强调

- **粗体**：使用 `**文字**`
- *斜体*：使用 `*文字*`
- ~~删除线~~：使用 `~~文字~~`

### 列表

无序列表：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

有序列表：

1. 第一步
2. 第二步
3. 第三步

### 链接和图片

链接：[GitHub](https://github.com)

图片：`![描述](图片链接)`

### 代码

行内代码：`code`

代码块：

```python
def hello():
    print("Hello, World!")
```

### 引用

> 这是一段引用文字。
> 可以有多行。

### 表格

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
| 1   | 2   | 3   |

## 进阶技巧

### 任务列表

- [x] 已完成的任务
- [ ] 未完成的任务

### 脚注

这是一段文字[^1]。

[^1]: 这是脚注内容

掌握 Markdown 能让你的写作效率大大提升，强烈推荐！
